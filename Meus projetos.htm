MEU PORTFÓLIO.

Gerenciador de Tarefas

Um aplicativo web simples que permite aos usuários criar, visualizar, atualizar e excluir tarefas. O sistema pode ter uma interface intuitiva onde o usuário pode adicionar novas tarefas, marcar tarefas como concluídas e filtrar tarefas por status (concluídas ou pendentes).

Calculadora de Economia de Energia Solar

Este projeto consiste no desenvolvimento de uma calculadora online de energia solar que permite aos usuários estimar o potencial de economia de energia elétrica ao instalar um sistema de energia solar em suas residências ou empresas. A calculadora fornece uma interface simples para inserir dados relevantes e calcula a economia financeira, o retorno do investimento, e ajuda a visualizar o impacto ambiental positivo da instalação de painéis solares.

Loja de Açaí, Cupuaçu e Sorvete

Este projeto consiste no desenvolvimento de um site para uma loja fictícia especializada na venda de açaí, cupuaçu e sorvete. O objetivo é criar uma plataforma simples e intuitiva, que permita aos usuários visualizar os produtos disponíveis, selecionar tamanhos e sabores, e realizar pedidos de maneira fácil e rápida.
